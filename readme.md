_A compact web page designed to assist you in taking quick notes._

## Details

Just HTML, JavaScript, and browser capabilities. So it can be stored and then used locally.

## Latest Release

Find the latest by following
**[Tabula](https://dnedzelskyi.github.io/tabula/dist/index.html)** link on GitHub Pages.

## Usage

Run locally

```bash
  npm i .
  npm run dev
```

Build a single-file bundle

```bash
  npm run build
```
